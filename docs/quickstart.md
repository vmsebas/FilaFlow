# Quick Start

Get FilaFlow running and add your first spool in 5 minutes.

## 1. Access the Dashboard

Open your browser and go to:
- **Local:** http://localhost:7912
- **Network:** http://YOUR_IP:7912

You'll see the modern mobile dashboard (or the classic view on desktop).

## 2. Add a Vendor (Optional)

Before adding filaments, you can register your preferred vendors:

1. Go to **Vendors** (or click the "Marcas" stat card)
2. Click **+ Create**
3. Enter vendor details:
   - Name: e.g., "Bambu Lab"
   - Website: https://bambulab.com

## 3. Add a Filament Type

Register the filament types you use:

1. Go to **Filaments** (or click "Filamentos" stat card)
2. Click **+ Create**
3. Fill in:
   - **Name:** e.g., "PLA Matte White"
   - **Vendor:** Select from dropdown
   - **Material:** PLA, PETG, ABS, etc.
   - **Color:** Pick from color picker
   - **Density:** Usually 1.24 g/cmÂ³ for PLA
   - **Diameter:** 1.75mm (standard)

## 4. Add Your First Spool

Now register an actual spool:

1. Click the **+** button (FAB) or go to **Spools**
2. Click **+ Create**
3. Fill in:
   - **Filament:** Select the type you created
   - **Initial Weight:** Usually 1000g (1kg spool)
   - **Used Weight:** How much you've used (0 for new)
   - **Location:** Optional, e.g., "Shelf A"
   - **Lot Number:** Optional, from spool label

## 5. Track Usage

After each print:

1. Find the spool in the dashboard
2. Click to edit
3. Update **Used Weight** with the amount consumed
4. The progress bar updates automatically
